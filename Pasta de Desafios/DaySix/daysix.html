Day Six - Remoção de Arrays

A remoção de arrays é semelhante à inserção. Entretanto, usamos novas formas de os remover. Podemos remover um item pelo primeiro índice da lista, pelo último ou do meio dela, através do index do array. O index é como referenciamos aquele item na lista. Sempre começamos pelo valor 0 até o valor do tamanho da lista menos um.
Por exemplo:
array = ['Banana', 'maca', 'estrogonofe']
Onde a posição de banana é [0], maca é [1], e estrogonofe é [2]. Sendo que o tamanho deste array é 3. Assim, podemos remover o seu primeiro item com .shift(), ou o seu último com .pop(). E qualquer outro com .splice(indice,1). Os itens serão realocados a devida nova posição, conforme a remoção feita.
Entretanto, é sempre necessário verificar como se dá a remoção de arrays na linguagem escolhida.
