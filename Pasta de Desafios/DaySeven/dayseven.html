Day Seven - Funções em JavaScript

Funções são partes do código onde podemos chamar mais de uma vez, sem a necessidade de escrevermos todas as linhas novamente no código. Elas podem ou não ter retorno de um valor. Por exemplo, se quisermos verificar mais de uma vez no código, se um número é ou não negativo, podemos criar uma função de verificaNumeroNegativo() e passar um parâmetro a ser recebido dentro dela com verificaNumeroNegativo(numero). Assim, podemos fazer nossa verificação do número dentro desta função. E darmos um retorno com base na condição satisfeita, com return "É negativo" ou return "É positivo". O return, serve para devolvermos um resultado à ação que chamou a função. Por exemplo:
let num1 = -1 e let num2 = 2. Podemos printar isto na tela com console.log(verificaNumeroNegativo(num1));
O resultado sairá com base no return que dermos ao nosso código dentro da função. O retorno pode ser de qualquer variável.
Vale ressaltar a importância de analisarmos a documentação de como uma função na nossa linguagem é definida. Em JavaScript é assim: function funcao ( ) { }.
