Day Five - Arrays e Coleções

Arrays e coleções, são formas de armazenarmos itens em listas. Um array é como se fosse uma lista de tarefas ou compras, uma coleção de dados referênciados por uma única variável. Normalmente é criada iniciando a variável e atribuindo ela à uma lista vazia - [ ] -, mas, é necessário verificar a documentação da linguagem para entender como criar um array (lista) na linguagem escolhida.
Ela então pode ser mexida com comandos para adicionar ou remover um item, na frente, no final ou em qualquer lugar. Também há maneiras de retornar o tamanho da lista para podermos utilizar em estruturas de repetição que servirão para realizar verificações em toda a lista.
